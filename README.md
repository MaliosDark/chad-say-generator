# Chad Say Generator

**Chad Say Generator** is a lightweight, browser-based tool that lets you generate Chad ASCII memes with your own custom messages. It brings the legendary meme energy of *Chad* into a terminal-style ASCII experience, but directly in your browser ‚Äî with style.

This project is fully static and built with plain HTML, CSS (Tailwind), and JavaScript. It runs without a server or backend, making it ideal for fun, low-resource meme generation.
